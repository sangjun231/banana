# Terraform + AWS 인프라 구축 가이드

## 📚 학습 목표

1. **Terraform 기초**: Infrastructure as Code (IaC) 개념 이해
2. **AWS 기초**: 클라우드 인프라 구조 이해
3. **보안 강화**: 방화벽, IAM, 네트워크 보안
4. **비용 관리**: Budget 설정 및 모니터링

## 🏗️ 전체 아키텍처

```
┌─────────────────────────────────────┐
│   Internet                          │
└──────────────┬──────────────────────┘
               │
┌──────────────▼──────────────────────┐
│   Application Load Balancer (ALB)  │  ← 외부 접근
│   - SSL/TLS (HTTPS)                 │
│   - 보안 그룹 (포트 443만 허용)      │
└──────────────┬──────────────────────┘
               │
┌──────────────▼──────────────────────┐
│   ECS Fargate Service                │  ← NestJS 서버
│   - Docker 컨테이너                 │
│   - 자동 스케일링                    │
└──────────────┬──────────────────────┘
               │
┌──────────────▼──────────────────────┐
│   VPC (Virtual Private Cloud)       │  ← 격리된 네트워크
│   - Public Subnet (ALB)              │
│   - Private Subnet (ECS)             │
│   - 보안 그룹 (방화벽 규칙)           │
└──────────────────────────────────────┘
```

## 🔒 보안 계층

1. **네트워크 레벨**
   - VPC로 격리
   - 보안 그룹 (방화벽)
   - Private Subnet 사용

2. **애플리케이션 레벨**
   - HTTPS 강제
   - Rate Limiting
   - 입력 검증

3. **인프라 레벨**
   - IAM 역할 (최소 권한)
   - Secrets Manager (환경 변수)
   - CloudWatch 모니터링

## 💰 예상 비용

- **ECS Fargate**: 월 ~$15-30 (최소 사양)
- **ALB**: 월 ~$16
- **ECR**: 무료 (저장소)
- **CloudWatch**: 무료 (기본)
- **총 예상**: 월 $30-50

## 📋 진행 순서

1. Terraform 기본 구조
2. AWS Provider 설정
3. VPC 및 네트워크
4. 보안 그룹 (방화벽)
5. IAM 역할
6. ECR (Docker 저장소)
7. ECS 클러스터
8. ECS 서비스
9. ALB (로드밸런서)
10. Budget 알림
